# MyDrugs Mod Wiki

This wiki is a **MkDocs-ready documentation site** generated from the Java and resource archives you uploaded.

The current snapshot documents a NeoForge Minecraft mod centered on:

- agriculture and harvesting for rye, malt, tobacco, cannabis, coca, and psychedelic mushrooms
- custom drug items, drinkable fluids, bottles, rolling mechanics, syringes, and handheld tools
- a large processing chain of manual and industrial machines
- an addiction / withdrawal / recovery simulation layered onto survival gameplay
- custom shaders, sounds, HUD feedback, and hallucination-style client effects
- a custom biome and world content for psychedelic mushroom generation

## What the mod currently contains

- **112 item language entries**
- **45 block language entries**
- **23 registered fluids** and **5 registered gases**
- **61 recipe JSON files** in the uploaded resource snapshot
- **13 shader programs** and **30 sound assets**

## Read this wiki in order

1. Start with **Getting Started** for the broad gameplay loop.
2. Use **Drugs**, **Machines**, and **Recipes & Production** for system-level details.
3. Use the **Appendices** for raw reference tables.

## Scope note

This wiki is based on the uploaded source and resource archives only. It documents the mod snapshot accurately, but it does **not** assume anything that was not present in those archives, such as external build files or unpublished datapacks.
